## Installation

> [!Note]
> I have not deployed this to the web yet so you will have to run this locally

First, clone the repository:
```bash
git clone https://github.com/JimmyTarson12/mypros-quotes.git
```
Next, CD into it:
```bash
cd mypros-quotes
```
Finally, install dependencies:
```bash
npm install
```
